# Creating Your First Hytale Server on RYLM

Ready to host your own Hytale server? This guide will walk you through every step of creating and launching your first server on RYLM.

## Prerequisites

Before you begin, make sure you have:

- A RYLM account (sign up at [rylm.net](https://rylm.net))
- A valid email address for account verification
- That's it! No credit card or payment information needed

## Step 1: Sign Up for RYLM

If you haven't already:

1. Go to [rylm.net](https://rylm.net)
2. Click **"Sign Up"** in the top right corner
3. Fill in your details:
   - Username
   - Email address
   - Password
4. Verify your email address
5. Log in to your new account

## Step 2: Navigate to Server Creation

Once logged in:

1. Click on the **"Create Server"** button on your dashboard
2. You'll be taken to the server creation wizard

## Step 3: Configure Your Server

### Basic Settings

**Server Name**
Choose a unique name for your server. This will be visible to players.

```
Example: "Epic Adventures Server"
```

**Server Type**
Select your server type:
- **Survival** - Classic Hytale survival experience
- **Creative** - Unlimited resources for building
- **Adventure** - Custom maps and quests
- **Minigames** - Party games and competitions
- **Custom** - Start with a blank slate

**Version**
Select your preferred Hytale version (we recommend using the latest stable release).

### Advanced Settings (Optional)

**Max Players**
Set the maximum number of players (1-10 for free tier).

```
Recommended: 5-8 players for optimal performance
```

**Game Mode**
- Survival
- Creative
- Adventure
- Spectator

**Difficulty**
- Peaceful
- Easy
- Normal
- Hard

**World Generation**
- Default Orbis
- Custom Seed
- Flat World
- Import World

## Step 4: Launch Your Server

1. Review your settings
2. Click **"Create Server"**
3. Wait 30-60 seconds for server initialization
4. Your server is now live! ðŸŽ‰

## Step 5: Connect to Your Server

### Your Server Address

After creation, you'll receive a unique server address:

```
your-server-name.rylm.net
```

### Connecting from Hytale

1. Launch Hytale
2. Click **"Multiplayer"**
3. Click **"Add Server"**
4. Enter your server details:
   - **Name**: Your server name
   - **Address**: your-server-name.rylm.net
5. Click **"Done"**
6. Select your server and click **"Join"**

## Step 6: Invite Your Friends

Share your server address with friends:

1. Copy your server address from the RYLM dashboard
2. Share via Discord, social media, or messaging
3. Friends can connect using the same steps above

### Generating an Invite Link

Use the built-in invite feature:

1. Go to your server dashboard
2. Click **"Invite Players"**
3. Copy the invite link
4. Share with your friends

## Managing Your Server

### Starting and Stopping

Control your server from the dashboard:

- **Start**: Click the green play button
- **Stop**: Click the red stop button
- **Restart**: Click the restart icon

**Note**: Servers automatically stop after 5 minutes of inactivity to save resources.

### Server Console

Access the live console to:

- View server logs
- Run commands
- Monitor player activity
- Debug issues

### Operator Permissions

Make yourself an operator:

1. Go to the **Console** tab
2. Type: `/op your-username`
3. Press Enter

You now have admin permissions!

## Essential Server Commands

Here are some useful commands to get started:

```bash
# Make a player operator
/op <username>

# Change game mode
/gamemode <mode> <player>

# Set time of day
/time set <day|night|value>

# Change weather
/weather <clear|rain|thunder>

# Teleport to a player
/tp <target> <destination>

# Set world spawn point
/setworldspawn

# Whitelist management
/whitelist add <player>
/whitelist remove <player>
```

## Troubleshooting

### Server Won't Start

- Check the console for error messages
- Verify your server settings
- Try restarting the server
- Contact support if issues persist

### Can't Connect to Server

- Verify the server address
- Make sure the server is running (green status)
- Check if you're using the correct Hytale version
- Disable any VPN or firewall that might block connections

### Performance Issues

- Reduce max player count
- Remove unnecessary plugins
- Optimize world settings
- Contact support for performance tips
