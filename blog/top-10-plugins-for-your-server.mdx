# Top 10 Must-Have Plugins for Your RYLM Server

Want to take your Hytale server to the next level? These essential plugins will help you create an amazing experience for your players. All are available for free installation on RYLM!

## 1. üõ°Ô∏è CoreProtect - Essential Grief Prevention

**Category**: Protection & Security

CoreProtect is the ultimate insurance policy for your server. It logs every block change, chest transaction, and player action, allowing you to rollback griefing with a single command.

### Key Features
- Complete block logging and rollback system
- Inspect any block to see its history
- Restore griefed areas in seconds
- Minimal performance impact
- Easy-to-use commands

### Why You Need It
Griefing happens. When it does, CoreProtect lets you fix it in minutes instead of hours.

### Quick Start
```bash
# Inspect a block
/co inspect

# Rollback a player's actions from the last hour
/co rollback u:PlayerName t:1h

# Restore (opposite of rollback)
/co restore u:PlayerName t:1h
```

**Install from**: RYLM Plugin Library ‚Üí Protection

## 2. üè† LandClaim - Protect Your Players' Builds

**Category**: Protection

Give players the power to protect their own creations with an intuitive land claiming system.

### Key Features
- Golden shovel claiming (or customizable item)
- Trust system for friends
- Automatic new player claims
- Prevent building, chest access, and PvP in claims
- Expandable claim sizes

### Why You Need It
Players want to build without fear. LandClaim ensures their hard work stays safe.

### Quick Start
```bash
# Players claim land by:
1. Get a golden shovel
2. Right-click two corners to create a claim
3. /trust <player> to allow friends access

# Admin commands
/claimlist - View all claims
/deleteclaim - Remove a claim
```

**Install from**: RYLM Plugin Library ‚Üí Protection

## Installing Multiple Plugins

Want to install several plugins at once?

1. Go to **Plugins ‚Üí Browse**
2. Use the checkbox on each plugin card
3. Click **"Install Selected"** at the bottom
4. Wait for batch installation
5. Restart your server

## Performance Considerations

More plugins = more load on your server. Tips for maintaining performance:

- Start with essentials (protection, economy, utilities)
- Add plugins gradually
- Monitor performance in your RYLM dashboard
- Disable unused plugin features
- Keep plugins updated
